MyFicDB is a self-hosted single-user app similar to AO3 for hosting Fan Fiction.  You can create multiple stories with chapters, assign tags, series and actors to each story.

## Features

### Current Features

- Story Creation with Chapter System and full WYSIWYG Editor
- Actor, Tag and Series Support
- Inline creation of Actors, Tags and Series during Story Creation
- No-app lock in, data is stored in SQLite and can be exported at any time
- HTML and Markdown export support for any Story
- Export all stories in HTML in a single click
- Fully self-hosted
- Light and Dark Mode
- Custom Styling/CSS Support

### Planned Features

These are planned features that I'd ideally like to add later on, that aren't implemented yet or are half implemented. 

- PDF Support for Exporting
- API (which would then allow a Mobile App to exist)
- Tag/Series Name Editing
- Additional properties for Actors, such as Eye Color, Species, etc.
